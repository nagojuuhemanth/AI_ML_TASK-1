Data Cleaning & Preprocessing (Titanic Dataset)

Objective
To clean and preprocess raw Titanic dataset data for Machine Learning.

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Steps Performed
- Downloaded dataset using kagglehub
- Checked missing values and data types
- Filled missing values (missing Age to median, missing Embarked to mode)
- Dropped Cabin column
- Encoded categorical variables (Sex, Embarked)
- Removed outliers using IQR method
- Applied Standardization
- Saved cleaned dataset as cleaned_titanic.csv

Output
A cleaned and standardized dataset ready for ML model training.
